# 控制流程语句

Java支持多种控制流程语句，包括条件语句、循环语句、跳转语句等。条件语句根据条件执行不同的代码块，循环语句重复执行一段代码，跳转语句可以跳出循环或者直接跳转到指定代码块。

```
cssCopy codeif (a > b) {
    System.out.println("a 大于 b");
} else {
    System.out.println("a 小于等于 b");
}

for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

while (a < b) {
    a++;
}

break; // 跳出循环
continue; // 跳过本次循环
```
